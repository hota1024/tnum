To install `tnum` in your project, run one of the following commands in your terminal.

<Tabs items={["npm", "pnpm", "yarn"]}>
  <Tab value="npm">
    ```shell allowCopy
    npm install tnum
    ```

  </Tab>
  <Tab value="pnpm">
    ```shell allowCopy
    pnpm install tnum
    ```

  </Tab>
  <Tab value="yarn">
    ```shell allowCopy
    yarn add tnum
    ```

  </Tab>
</Tabs>
